## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Consultor logado -> rota /adocao | Browser pode navegar diretamente; gate UX via `RequireRole` + RLS da Wave 1 garante que mesmo bypass do RequireRole nao expoe dados de outras tabelas |
| Consultor logado -> queries de adocao (audit_logs, leads, tarefas, perfis) | RLS aplica `is_interno()`/`current_cliente_id()` policies — interno ve tudo, cliente nao acessa nenhuma |
