## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SW postMessage → AppLayout navigate | URL string vindo do SW pode ser atacker-controlled se payload original o foi |
| BeforeInstallPromptEvent → `prompt()` | Evento global window; libs externas teoricamente podem hijackear se compartilharem window |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-07 | Tampering | postMessage PUSH_NAVIGATE com URL externa → open redirect via React Router | mitigate | AppLayout listener (Task 5.2) valida `new URL(e.data.url).origin === window.location.origin` antes de navigate. Defense-in-depth: SW já valida (Plan 02 Task 2.2). |
| T-06-20 | Tampering | URL malformada no postMessage causa throw em URL constructor | mitigate | Task 5.2 wraps `new URL(...)` em try/catch com warn log; ignora message inválida em vez de crashar AppLayout. |
